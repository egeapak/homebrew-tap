# Homebrew Tap

Homebrew formulae for [egeapak](https://github.com/egeapak) projects.

## Installation

```bash
brew tap egeapak/tap
```

## Available Formulae

| Name | Type | Description | Version |
|------|------|-------------|---------|
| [mergers](https://github.com/egeapak/mergers) | Formula | CLI/TUI tool for streamlining Azure DevOps PR merging via cherry-picking | 1.0.0 |
| [engramdb](https://github.com/egeapak/engramdb) | Cask | Project-scoped persistent memory store for coding agents | 0.3.0 |

### mergers

```bash
brew install mergers
```

### engramdb

Requires `gh auth login` (private repository).

```bash
brew install --cask engramdb
```
